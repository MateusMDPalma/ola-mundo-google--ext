Sobre

Extensão Chrome de exemplo (Manifest V3) que exibe uma mensagem e um botão com animações, acessível e com dark mode automático.

Baixar .zip da extensão Ver no GitHub

Web demo do popup

Olá, mundo!

Clique para alternar a mensagem e a cor.

Como instalar localmente

  1. Baixe o arquivo dist/ola-mundo-ext-v1.1.3.zip e descompacte.
  2. Abra chrome://extensions/ no Chrome.
  3. Ative o Modo do desenvolvedor.
  4. Use Carregar sem compactação e selecione a pasta da extensão (a que contém manifest.json).
