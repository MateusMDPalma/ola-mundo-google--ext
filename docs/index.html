Sobre

Extensão Chrome (Manifest V3) que exibe uma mensagem e um botão com animações, acessível e com dark mode automático.

Baixar .zip da extensão Ver no GitHub

Se preferir, publique uma Release e troque o link do .zip por um link da Release.

Web demo do popup

Olá, mundo!

Clique para alternar a mensagem e a cor.

Instalação local

  1. Baixe dist/ola-mundo-ext-v1.1.3.zip e descompacte.
  2. Abra chrome://extensions/ no Chrome.
  3. Ative Modo do desenvolvedor.
  4. Use Carregar sem compactação e selecione a pasta (a que contém manifest.json).
